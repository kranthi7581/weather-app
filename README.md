## 📌 Overview

This Weather App is a **responsive web application** built using **HTML, CSS, and JavaScript**.
It allows users to search for any city and view real-time weather details using **fake weather APIs**.

The project focuses on simplicity, clean UI, and providing a smooth user experience across all devices.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Displays temperature, weather condition, and location
* 📱 Fully responsive design (Mobile, Tablet, Desktop)
* ⚡ Fast and lightweight performance
* 🎨 Clean and user-friendly interface
* 🔄 Dynamic data rendering using JavaScript

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling & responsiveness
* **JavaScript (ES6)** – Logic & API handling
* **Fake Weather API** – Data simulation
